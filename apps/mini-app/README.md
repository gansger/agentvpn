# Telegram Mini App

Responsive frontend реализует публичный сайт и Telegram Mini App по макетам Stitch из
`docs/STITCH_DESIGN.md`.

- На desktop отображается публичная витрина для пользователей и модерации платёжной
  системы: описание услуги, тарифы, порядок подключения, поддержка и правовые разделы.
- На mobile и внутри Telegram отображается Mini App с экранами тарифов, оформления
  платежа через СБП, проверки оплаты, подключения и профиля.
- Frontend использует существующие backend endpoints и не содержит секретов.

Служебный ENOT meta-тег в `public/index.html` удалять нельзя.
